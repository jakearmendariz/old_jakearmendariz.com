    Games

    LightSpeed

    Lightspeed is fairly similar to Asteroid. I began and finished the game in July 2018. To play use the arrow keys to move rocket across screen avoiding asteroids, planets and suns. Users can shoot laser bullets at objects, gray objects (asteroids) are one bullet, blue (planets) take two hits and yellow (suns) take three hits.

    Flappy Plane

    Flappy Plane is just flappy bird with a plane, it is not original. Rather, it was my first attempt to use my 'library'. I ended up having to tweek a little too much of the original program. So now I know to focus more time on library construction and reliablity before using it for other games.

    Snakers

    Snakers. It is the first game I created in June 2018, it's really just the game snake with three levels of difficulty. Each level increases the speed of the game along with adding other ways to die. I think I will beef this one up a little more this summer so that it is not just a copy of the original game.

    Game Library

    I created this basic 'game' in early 2019 in order to create a basic library for myself to use when creating other games. I programmed basic collisions, gravity and keyboard interactivity in hopes of using it to build games in the future.
