    Projects

    Snake AI

    The snake game, but an AI plays it. Built in Java-Processing using a slight modification of A* for pathfinding, a min-heap class for higher efficiency and ran on processing. If you'd like to see it in action, I created a youtube playlist displaying it, here.

    Charity Cloud

    In the Santa Cruz 2020 hackathon I created a website called Charity Cloud. I worked the backend, using Flask (a micro web framework written in python) and mongoDB to create charities in our system, then display them on the site. Using my knowledge of HTML/CSS to render them dynamically. We are still working on hosting this website, but in the mean time you can check out the dev post here.

    A* Simulator

    A* made in Java-Processing. Finds the shortest path between two points on 2D grid, displays path, obstacles and vision. Has ability to add, erase obstacles, change map size, unit size, start and end point.

    List ADT

    An ADT built in C that works as a doubly linked list with a cursor that can insert nodes, expandable to carry different information. Lex.c class intakes a series of lines and sorts them alphabetically using the List ADT.

    Big Integer ADT

    An ADT using List.c to handle big integer operations in the form of thousands of digits in less than a few seconds. Intakes longs as input files of numbers, turns into List ADT of long integers of undefinded length, then can add, multiply, subtract and divide.

    Matrix ADT

    Sparse Matrix Calculator: A complete ADT constructed of an array of linked lists that can be manipulated as a 2-D matrix, but with much higher efficiency. Can add, subtract, multiply and print Matrices of enormous size.

    Dijkstra's Algorithm

    Preforms Dijkstra's algorithm on Lists of nodes and their edges. Utilizes List ADT to create graphs, search through them and find distance between nodes.

    Linear Algebra Tutor

    Built with Javscript, HTML, CSS. This is a simple, responsive problem solving and explaining tool. Not fully implemented dictionary, but ability to solve multiple linear algebra problems.

    LightSpeed

    A game I created before college. Written in javascript, you fly around as a rocket shooting asteroids, press space to shoot, use arrow keys to move around screen.
